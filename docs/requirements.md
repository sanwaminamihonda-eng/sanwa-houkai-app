# 要件定義

## 機能要件

### 訪問記録
- 訪問記録の入力・編集・削除
- 一覧表示・検索
- AI要約生成

### スケジュール
- 日/週/月ビュー表示
- 作成・編集・削除
- 繰り返し予定
- 職員別/利用者別フィルタ

### 利用者管理
- 利用者情報の登録・編集
- 要介護度、ケアプラン管理
- 一覧・検索

### 職員管理
- 職員情報の登録・編集
- 資格情報管理

### レポート
- 月次サービス提供実績
- 利用者別/職員別実績
- PDF出力

### 認証
- Google OAuth
- Email/Password
- 管理者/職員の権限分離
- 事業所単位のデータ分離

## 非機能要件

| 項目 | 要件 |
|------|------|
| パフォーマンス | 画面表示3秒以内、API応答1秒以内 |
| 可用性 | 99.5%以上 |
| セキュリティ | HTTPS、個人情報保護、アクセスログ |
| 運用 | 日次バックアップ、エラー監視、CI/CD |
