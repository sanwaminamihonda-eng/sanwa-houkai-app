# 画面仕様

## 画面一覧

| 画面 | パス | 説明 | 優先度 |
|------|------|------|--------|
| ログイン | `/login` | 認証画面 | P0 |
| 記録入力 | `/records/new` | 訪問記録の新規作成 | P0 |

| 履歴一覧 | `/records` | 訪問記録の一覧 | P0 |
| 履歴詳細 | `/records/:id` | 訪問記録の詳細表示 | P0 |
| 利用者一覧 | `/clients` | 利用者リスト | P1 |
| 利用者詳細 | `/clients/:id` | 利用者情報詳細 | P1 |
| スケジュール | `/schedule` | カレンダー表示 | P1 |
| 報告書一覧 | `/reports` | 実施報告書リスト | P2 |
| 計画書一覧 | `/plans` | 訪問介護計画書リスト | P2 |
| 設定 | `/settings` | アプリ設定 | P2 |

## ナビゲーション構成

### ボトムナビ（モバイル）

```
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│  記録   │  履歴   │ 予定表  │ 利用者  │  その他  │
│   +     │   ⏱    │   📅    │   👤    │   ⋮    │
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

| タブ | アイコン | 遷移先 |
|------|---------|--------|
| 記録 | add / edit | 記録入力画面 |
| 履歴 | history | 履歴一覧 |
| 予定表 | calendar | **スケジュール管理（Googleカレンダー風）** |
| 利用者 | person | 利用者一覧 |
| その他 | more | 設定・帳票メニュー |

※ 予定表の詳細仕様は [schedule-feature.md](./schedule-feature.md) を参照

## P0画面詳細

### 記録入力画面

**目的**: 訪問介護記録の入力

**レイアウト（モバイル）**:

```
┌────────────────────────────────┐
│ ← 記録入力            保存     │ ← ヘッダー
├────────────────────────────────┤
│                                │
│ ┌────────────────────────────┐ │
│ │ 利用者 *                   │ │ ← ドロップダウン
│ │ [選択してください     ▼]   │ │
│ └────────────────────────────┘ │
│                                │
│ ┌────────────────────────────┐ │
│ │ 訪問日 *        訪問理由   │ │
│ │ [2026/01/09]    [定期  ▼]  │ │
│ └────────────────────────────┘ │
│                                │
│ ┌────────────────────────────┐ │
│ │ 開始時間 *      終了時間 * │ │
│ │ [09:00]         [10:00]    │ │
│ └────────────────────────────┘ │
│                                │
│ ── バイタル ──────────────────│
│                                │
│ ┌──────┬──────┬──────────────┐ │
│ │ 脈拍 │血圧高│ 血圧低       │ │
│ │ [72] │[120] │ [80]         │ │
│ └──────┴──────┴──────────────┘ │
│                                │
│ ── サービス内容 ──────────────│
│                                │
│ ┌────────────────────────────┐ │
│ │ 身体介護                   │ │
│ │ ☑ 排泄介助  ☑ 食事介助    │ │
│ │ ☐ 入浴介助  ☐ 更衣介助    │ │
│ └────────────────────────────┘ │
│                                │
│ ┌────────────────────────────┐ │
│ │ 生活援助                   │ │
│ │ ☐ 掃除  ☐ 洗濯  ☐ 調理   │ │
│ └────────────────────────────┘ │
│                                │
│ ── 特記事項 ──────────────────│
│                                │
│ ┌────────────────────────────┐ │
│ │ [AI生成] ボタン            │ │
│ ├────────────────────────────┤ │
│ │                            │ │
│ │ (テキストエリア)           │ │
│ │                            │ │
│ └────────────────────────────┘ │
│                                │
│         [保存]                 │
│                                │
└────────────────────────────────┘
```

**入力項目**:

| 項目 | 型 | 必須 | 備考 |
|------|-----|------|------|
| 利用者 | Select | ✓ | 利用者リストから選択 |
| 訪問日 | DatePicker | ✓ | デフォルト:今日 |
| 訪問理由 | Select | | 定期/臨時/その他 |
| 開始時間 | TimePicker | ✓ | |
| 終了時間 | TimePicker | ✓ | |
| 脈拍 | Number | | 単位: bpm |
| 最高血圧 | Number | | 単位: mmHg |
| 最低血圧 | Number | | 単位: mmHg |
| 身体介護 | MultiSelect | | チップ形式で複数選択 |
| 自立支援 | MultiSelect | | |
| 生活援助 | MultiSelect | | |
| 特記事項 | TextArea | | AI生成オプション |
| 画像 | ImagePicker | | 複数添付可 |

**アクション**:
- 保存: バリデーション後にDB保存
- AI生成: Geminiで特記事項を自動生成
- キャンセル: 確認ダイアログ後に戻る

### 履歴一覧画面

**目的**: 訪問記録の一覧表示・検索

**レイアウト（モバイル）**:

```
┌────────────────────────────────┐
│ 履歴                    🔍 ⚙  │
├────────────────────────────────┤
│ [利用者で絞り込み        ▼]   │ ← フィルタ
├────────────────────────────────┤
│                                │
│ ┌────────────────────────────┐ │
│ │ 2026/01/09 (木) 09:00-10:00│ │ ← カード
│ │ 山田 太郎                  │ │
│ │ 身体介護: 排泄、食事       │ │
│ │ 担当: 佐藤                 │ │
│ └────────────────────────────┘ │
│                                │
│ ┌────────────────────────────┐ │
│ │ 2026/01/08 (水) 14:00-15:00│ │
│ │ 鈴木 花子                  │ │
│ │ 生活援助: 掃除、洗濯       │ │
│ │ 担当: 佐藤                 │ │
│ └────────────────────────────┘ │
│                                │
│ (スクロール)                   │
│                                │
└────────────────────────────────┘
```

**機能**:
- 日付降順で表示（デフォルト）
- 利用者/日付/担当者でフィルタ
- 無限スクロール or ページネーション
- タップで詳細画面へ
- スワイプで編集/削除（オプション）
